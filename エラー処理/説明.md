# エラーのキャッチ

ルート・ハンドラとミドルウェアによって呼び出された非同期関数から返されたへらーについては、それらをnext()に渡す必要がある。


```javascript
app.get('/', function (req, res, next) {
  fs.readFile('/file-does-not-exist', function (err, data) {
    if (err) {
      next(err) // Pass errors to Express.
    } else {
      res.send(data)
    }
  })
})
```
